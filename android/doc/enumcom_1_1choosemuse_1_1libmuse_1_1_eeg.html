Detailed Description

Represents the data mapping in a EEG packet and in a Quantization packet. Enum values correspond to head locations. Check http://developer.choosemuse.com/hardware-firmware/hardware-specifications for details.

Raw EEG values are given in microvolts. EEG derived values may have different units as defined in MuseDataPacketType .
